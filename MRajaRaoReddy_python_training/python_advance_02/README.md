# Python Advanced Training - Assignment 2

## Overview
This folder contains advanced Python assignments focusing on data manipulation, visualization, and analysis using popular data science libraries.

## Prerequisites
- Python 3.7+
- pip (Python package manager)

## Installation

Install all required dependencies using the `requirements.txt` file:

```bash
pip install -r requirements.txt
```

### Required Libraries
- **NumPy** - Numerical computing library for array operations
- **Pandas** - Data manipulation and analysis library
- **Matplotlib** - Data visualization library
- **Seaborn** - Statistical data visualization library built on Matplotlib

## Assignments

### Assignment 1: NumPy Basics
**File:** `assignment_01.ipynb`

**Topics Covered:**
- Creating NumPy arrays
- Basic array operations (mean, max, min, sum)

- Matrix creation and reshaping

**Learning Outcomes:**
- Understand NumPy array structure and properties
- Perform basic mathematical operations on arrays
- Work with multi-dimensional arrays and matrices
